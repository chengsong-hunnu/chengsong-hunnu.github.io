掘金的总结

1.箭头函数

箭头函数的作用主要是定义匿名函数

let foo = function () {};
// 转换成箭头函数
let foo = () => {};

基本语法（参数）：

  • 匿名函数没有参数：() 不能省略，占位作用。let foo = () => {};
  • 只有一个参数：() 可以省略，也可以不省略。let foo = a => {};
  • 多个参数，() 不能省略。let foo = (a,b) => {};

箭头函数的特点：

1、不能做构造函数（不能实例化）

2、没有 arguments。 arguments就是函数带的实参，类似linux的argv，可用arguments获取调用的参数。

3、箭头函数没有自己的this，箭头函数的this不是调用的时候决定的，this指向上下文环境。（意思：箭头函数的外层的是否有函数，如果有，箭头函数的this就是外层函数的this，如果没有，则为 window）。是由其定义的时候的上下文决定的，是固定的

2.let 与 const

let 声明的变量只在 let 命令所在的代码块内有效。

const 声明一个只读的常量，一旦声明，常量的值就不能改变。const其实是让这个变量指向的地址数据不能发生变化，当变量是普通类型的时候，就不能变。当变量是数组，对象这些引用类型的时候，因为这个地址只是指向实际数据的指针，所以可以改变这些引用类型里面的数据，只是const定义的变量不能改变。

3.解构赋值

解构赋值是对赋值运算符的扩展，他是一种针对数组或者对象进行模式匹配，然后对其中的变量进行赋值。

可进行嵌套解构。

从对象或数组中提取数据, 并赋值给变量(多个)。

注意当对对象进行解构的时候，在{}里面的变量名需要与被解构的对象key名相同，不然不能被解构。

let obj = {name : 'kobe', age : 39};
let name = obj.name;
let age = obj.age;
//解构
let {name,age} = obj;
let [a,b,c] = [1,2,3];
console.log(name,age)
//注意这种方式，是对常见等号赋值的一种对称形式    source --> target
let {name:x,age:y} = obj;
console.log(name,age)  //referrence error
console.log(x,y)  //‘kobe’ 39

4.展开符/三点运算符(…)/rest参数

用来取代 arguments 但比 arguments 灵活.里面的value是一个真正的数组，而不是像argument一样的伪数组

而且只能是函数参数的最后一个位置，即后面不能有其他的参数

或者可用来对数组进行扩展

let arr = [1, 6];
let arr1 = [2, 3, 4, 5];
arr = [1, ...arr1, 6];
f(...arr1);
console.log(arr); // [1,2,3,4,5,6]
console.log(...arr); // 1 2 3 4 5 6

5.async 函数

async 函数返回一个 Promise 对象，可以使用 then 方法添加回调函数。

async function helloAsync(){
    return await "helloAsync";
  }
console.log(helloAsync())  // Promise {<resolved>: "helloAsync"}
helloAsync().then(v=>{
   console.log(v);         // helloAsync。async里的return返回会成为then()回调函数的参数。
})

async 函数中可能会有 await 表达式，async 函数执行时，如果遇到 await 就会先暂停执行 ，等到触发的异步操作完成后，恢复 async 函数的执行并返回解析值。

6.class类

为了方便创建对象。形似java里面的class类

//定义类
class Point {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }
  toString() {
    return '(' + this.x + ', ' + this.y + ')';
  }
}
let newclass = new Point(x,y);//创建

constructor方法是类的默认方法，通过new命令生成对象实例时，自动调用该方法。一个类必须有constructor方法，如果没有显式定义，一个空的constructor方法会被默认添加。

class可以继承。extends

class ColorPoint extends Point {
  constructor(x, y, color) {
    super(x, y); // 调用父类的constructor(x,y)，子类必须在constructor方法中调用super方法，主要构建父类的this对象。
    this.color = color;
  }
  toString() {
    return this.color + ' ' + super.toString(); // 调用父类的toString()
  }
}

1.模板字符串

使用 `` 这两个反引号字符里面包围的字符串称为模板字符串，使用 ${} 可使用正常作用域里面可使用的变量，函数

let x = 1;
let y = 2;

`${x} + ${y} = ${x + y}`
// "1 + 2 = 3"

`${x} + ${y * 2} = ${x + y * 2}`
// "1 + 4 = 5"

let obj = {x: 1, y: 2};
`${obj.x + obj.y}`

2.尾调用，尾递归

在函数的return后面加一个函数，可以优化调用栈，在严格模式下启用
